Open-Source Software

We develop and maintain open-source software to make our research methods accessible to the broader scientific community. All code is released under permissive licenses (MIT/Apache) and accompanied by documentation, tutorials, and examples.

  • Foam-Agent — End-to-end, composable multi-agent framework for automating CFD simulations in OpenFOAM. (NeurIPS 2025 Workshop)
  • Deep Autoencoder with SVD Convergence — Learnable weighted dimensionality reduction frameworks for fluid dynamics.
  • DiagramBank — A dataset of diagram design exemplars with paper metadata for retrieval-augmented generation.
  • FoamGPT — Dataset and tools for LLM-based computational fluid dynamics.
  • pykoopman — A Python package for data-driven approximation of the Koopman operator. Published in the Journal of Open Source Software (2023).
  • Neural Implicit Flow (NIF) — A library for mesh-agnostic dimensionality reduction of spatiotemporal PDEs.
