# Font Files Directory

Place your "The Very Vogue Serif" font files in this directory.

## Required Font Files

The CSS expects these font files (in order of preference):
- `TheVeryVogueSerif-Regular.woff2` (or `.woff` or `.ttf`)
- `TheVeryVogueSerif-Bold.woff2` (or `.woff` or `.ttf`)
- `TheVeryVogueSerif-Italic.woff2` (or `.woff` or `.ttf`)
- `TheVeryVogueSerif-BoldItalic.woff2` (or `.woff` or `.ttf`)

## Font File Formats

- **WOFF2** (preferred) - Best compression and browser support
- **WOFF** - Good compression, wide browser support
- **TTF** - Fallback format, larger file size

## Where to Get the Font

"The Very Vogue Serif" is available from:
- FontSpring: https://www.fontspring.com/fonts/nicky-laatz/very-vogue
- Creative Market: Search for "Very Vogue Serif"

## Note

If your font files have different names, you can either:
1. Rename them to match the expected names above, OR
2. Update the file paths in `utils/css_layout.css` to match your actual file names
